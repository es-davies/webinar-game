## GridLands Prompt
Think before you build.
Build a simple hex tile placement game as a single HTML file.

## Game:
Start with one tile in the center.
Arrange tiles in a honeycomb grid so edges touch.
The map grows outward as the player places tiles.
A tile can only be placed on an EMPTY spot that touches an existing tile.
Show these valid spots as faint "ghost hexes".
Clicking a ghost hex places the next tile.
Limit the game to 40 placed tiles, then show "All done"
Left sidebar with title, tiles remaining, New Game button, and tile legend.
Bottom-right "Next Tile" preview.

## Tiles (random next tile each turn):
Forest
Village
Field
River
Rail
